# ROS 2 Tank Robot Learning Project

Learning autonomous robotics through hands-on projects using ROS 2, Raspberry Pi 5 and mobile robot platforms.

## Current Focus
- ROS 2 fundamentals
- Python for robotics
- Sensor integration
- Autonomous mobile robots
- Computer vision
- Robotics networking

## Hardware
- Raspberry Pi 5
- Yahboom ROS Expansion Board V3.0
- OAK-D Lite FF
- BNO085 IMU
- Tank robot platform with encoder motors

## Planned Features
- ROS 2 teleoperation
- Sensor visualization in RViz
- LiDAR integration
- SLAM and mapping
- Autonomous navigation
- Multi-robot communication experiments

## Goals
This repository documents my robotics learning journey through real-world projects and experimentation with autonomous systems, UGVs and robotics software architectures.

## Repository Structure

- docs/    -> project notes and documentation
- images/  -> photos and diagrams
- videos/  -> robot demos
- src/     -> source code
